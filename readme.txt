== Description ==
Convert all posts (or a specific post type) from Classic to Gutenberg editor format with the click of a button.


== Installation ==
1. Install the plugin through the WordPress plugins page.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Take a back up of the pages that you wish to convert (e.g. using WordPress Importer or UpdraftPlus plugin).
3. Use the Classic to Gutenberg Converter screen to convert posts of all types or a specific type.


== Frequently Asked Questions ==

= How do I use this plugin? =
After activating the plugin, go to the Classic to Blocks Converter page in the admin menu. Select the post type you want to convert and click "Convert All Posts".
